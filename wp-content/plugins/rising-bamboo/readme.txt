== Installation ==
How to install
Important: You need to install this Plugin before installing the theme.
There are 2 different ways to install Rising Bamboo Core.
1 - Uploading in WordPress Dashboard
Navigate to the ‘Add New’ in the plugins dashboard
Navigate to the ‘Upload’ area, Select the zip file (The Plugin is included in the theme package you download from Themeforest after unpacking) from your computer →  Click ‘Install Now’
Activate the Plugin in the Plugin dashboard

2 - Using FTP
The Plugin is included in the theme package you download from Themeforest after unpacking
Unzip the zip file, which will extract the rising-bamboo directory to your computer
Upload the rising-bamboo directory to the /wp-content/plugins/ directory in your web space   
Activate the plugin in the Plugin dashboard
